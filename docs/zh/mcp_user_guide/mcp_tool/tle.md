# TLE Kernel

您可以使用 MCP 工具在 NVIDIA 上生成带有 TLE 的 Kernel。本节以 sparse\_mla Kernel 为例。有关该 Kernel 的详细信息，请参阅 [FlagTree 文档](https://jwolpxeehx.feishu.cn/wiki/DcB6wnUlyiJzaHkmlUQcUNfcnpb#share-U0vAdwqqooUphDxTAWocWyXJnad)。

**前提条件**：在生成 TLE 算子之前，请预先安装 FlagTree 3.6.x 分支版本。请参阅 [安装 FlagTree](https://docs.flagos.io/projects/FlagTree/en/latest/getting_started/install.html)。

**示例**：与 AI 智能体对话，描述您生成 ReLU 算子的需求。

- **典型需求**：包含"调用 MCP 工具"（必填）、算子名称（必填）和任务描述（必填）。

- **需求示例**："**调用 MCP 工具生成 TLE ReLU 算子**。"

**注意**：TLE 算子生成功能为实验性功能，目前正在积极开发中。
